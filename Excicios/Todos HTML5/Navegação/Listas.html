Para listas definimos como ul e /ul para listas não ordenadas como as anteriores; ol e /ol usada para listas ordenadas como:

  1. Acordar
  2. Levantar
  3. Escovar os dentes
  4. Colocar a roupa
Lembrando que colocamos li e /li dentro da ol ou ul para definirmos aquilo como pertecente à uma lista. Também podemos definir tipos dentro da lista ao usar li type na ol:
  1. 'a' indica letras minúsculas,
  2. 'A' indica letras maiúsculas,
  3. 'i' indica algarismos romanos minúsculos,
  4. 'I' indica algarismos romanos maiúsculos,
  5. '1' indica números (padrão)
Para as ul temos como colocar ul dentro de outras ul's.
  • círculo
    • disco
      • quadrado
